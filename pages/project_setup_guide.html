Diagram Wiring Kebun Pintar

ESP32 DevKit V1:

  • • 3.3V → VCC (Relay)
  • • GND → GND (Common)
  • • GPIO2 → Control (Relay)
  • • A0 → LDR + Resistor 10kΩ

Relay 5V:

  • • VCC → 3.3V ESP32
  • • GND → Ground
  • • IN → GPIO2
  • • NO/COM → AC Line (Lampu)

Keamanan Listrik:

Pastikan instalasi AC dilakukan oleh teknisi berpengalaman. Gunakan MCB/sekering untuk proteksi rangkaian.

Setup Software Kebun Pintar

Instalasi Arduino IDE, library, dan konfigurasi kode untuk sistem monitoring kebun dengan automasi

1

Install Arduino IDE

Download dan install Arduino IDE versi terbaru dari website resmi Arduino.

Download: https://www.arduino.cc/en/software

Download IDE
2

Setup ESP32 Board

Tambahkan ESP32 board manager ke Arduino IDE.

File → Preferences → Additional Board Manager URLs:

https://dl.espressif.com/dl/package_esp32_index.json

Kemudian: Tools → Board → Boards Manager → Cari "ESP32" → Install

3

Install Library

Install library yang diperlukan untuk LDR sensor dan MQTT.

Library Manager → Install:

  • • WiFi by Arduino
  • • PubSubClient by Nick O'Leary
  • • ArduinoJson by Benoit Blanchon

Troubleshooting

Solusi untuk masalah umum yang mungkin Anda temui selama setup

ESP32 Tidak Terdeteksi

Board ESP32 tidak muncul di Arduino IDE atau tidak bisa di-upload.

Solusi:

  • • Install driver CP2102 atau CH340
  • • Tekan tombol BOOT saat upload
  • • Cek kabel USB (gunakan kabel data)
  • • Pilih port COM yang benar

WiFi Tidak Terhubung

ESP32 gagal connect ke jaringan WiFi.

Solusi:

  • • Pastikan SSID dan password benar
  • • Gunakan WiFi 2.4GHz (bukan 5GHz)
  • • Cek jarak dari router
  • • Restart ESP32 dan router

Sensor Tidak Terbaca

BH1750 tidak memberikan data atau nilai selalu 0.

Solusi:

  • • Cek wiring SDA/SCL
  • • Pastikan power 3.3V stabil
  • • Scan I2C address (0x23 atau 0x5C)
  • • Ganti sensor jika rusak
